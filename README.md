# BMA_Quantencomputer

## Berufsmaturitätsarbeit: Quantencomputer

Dieses Repository enthält unsere Berufsmaturitätsarbeit zum Thema **Quantencomputer**.

### Inhalt

Die Arbeit behandelt folgende Themen:
- Grundlagen der Quantencomputer
- Datenverschlüsselung im Kontext von Quantencomputing
- Fazit und Ausblick auf zukünftige Entwicklungen

### Struktur

- `main.tex` – Hauptdokument der Arbeit
- `sections/` – Einzelne Kapitel der Arbeit
- `images/` – Abbildungen und Grafiken
- `literatur.bib` – Literaturverzeichnis

### Kompilierung

Das Dokument kann mit LaTeX kompiliert werden:
```bash
pdflatex main.tex
biber main
pdflatex main.tex
pdflatex main.tex
```
